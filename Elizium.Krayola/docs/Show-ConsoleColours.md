# Show-ConsoleColours

## SYNOPSIS

  Helper function that shows all the available console colours in the colour
  they represent. This will assist in the development of colour Themes.

## SYNTAX

```powershell
Show-ConsoleColours
```

## DESCRIPTION

Shows console colours.

## PARAMETERS

## INPUTS

### None

## OUTPUTS

### System.Object
